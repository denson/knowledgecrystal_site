# Ariadne Thread — Onboarding

An interactive skill that introduces Ariadne Thread to anyone — developers who want to deploy it, agent builders who need document memory, or people evaluating whether it fits their needs.

## How to Launch

In Claude Code or Cowork, with the project folder selected:

> **Execute the SKILL.md file in ariadne-thread-onboarding**

Or trigger it with any of these phrases:

> **"What is Ariadne Thread?"**
> **"Walk me through Ariadne Thread"**
> **"How do I get started with Ariadne Thread?"**
> **"What deployment options are there?"**

The skill adapts to what you're trying to do and walks you through it one step at a time.

## What It Does

1. Opens with an accessible hook explaining why Ariadne Thread exists (with a visual)
2. Asks what you're trying to do
3. Routes to a needs-specific walkthrough with illustrations at each step
4. Paces itself — one idea and one image at a time, waiting for you to respond

## Audiences Supported

- **Developer** — Deploy to Railway and connect Claude Code in ~5 minutes
- **Agent builder** — Connect Open Brain, OpenClaw, or custom agents via MCP
- **Evaluator** — Understand architecture, costs, format support, and limitations
