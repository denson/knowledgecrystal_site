# Research Explainer

An interactive skill that explains the "Structured Data as LLM Tools" research to any audience — from curious non-technical people to engineers, researchers, and business leaders.

## How to Launch

In Claude Cowork, with the project folder selected, say:

> **Execute the SKILL.md file in research-explainer**

Or if you've opened Cowork directly in this directory:

> **Execute the SKILL.md file in the project root**

The skill will walk you through the research one step at a time, adapting to your background and interests.

## What It Does

1. Opens with an accessible hook that works for anyone (with visual diagrams)
2. Asks who you are and what you care about
3. Routes to an audience-specific deep dive with figures, evidence, and domain-relevant examples
4. Paces itself — one idea and one image at a time, waiting for you to respond before continuing

## Audiences Supported

- **General / Curious** — No technical background needed
- **Business / Decision-maker** — Tool investment, team composition, process design
- **Professional Practitioner** — Lawyers, accountants, actuaries, underwriters, compliance officers
- **Engineer / Practitioner** — Technical architecture, code analysis, methodology adaptation
- **Researcher / Academic** — Methodology, experimental design, collaboration opportunities
- **Nate B. Jones Community** — Skills ecosystem, specialist stacks, knowledge work gaps
